# Install corenote
$ brew tap AungMyoKyaw/tap

Capabilities
Everything you need,
nothing you don't
A focused tool that does one thing well — gives you full control of Apple Notes from your terminal.

Full CRUD

List, show, create, edit, delete, and move notes. Everything Notes.app does, from the command line.

Folder management

Create, rename, delete, and nest folders. Full organizational control, including nested hierarchies.

Fuzzy matching

Find notes by approximate title. Four-level matching: exact, prefix, contains, and Levenshtein distance.

Markdown in, Markdown out

Write in Markdown — corenote converts to and from Apple Notes' internal protobuf format transparently.

$EDITOR integration

Create and edit notes in your preferred editor. Works with vim, nvim, nano, or whatever you set.

JSON output

Every query supports --json for structured output. Pipe to jq, scripts, or automation.

Commands
Eight commands.
Zero complexity.
Each command is designed to do exactly what you expect, with sensible defaults and powerful options when you need them.

List all notes

Browse your notes with optional filtering by folder or account, custom sorting, and pagination. Perfect for getting an overview.

--folder --account --limit --sort --json
$ corenote list
$ corenote list --folder Work
$ corenote list --sort title --json
$ corenote list --limit 10

Show a note

Display a note's full content by fuzzy-matching its title. Use --raw for plain text or --id for direct lookup.

--id --raw --json
$ corenote show "meeting notes"
$ corenote show 42 --id
$ corenote show "my note" --raw

Search notes

Full-text search across all note titles and content. Optionally scope to a specific folder.

--folder --limit --json
$ corenote search "project deadline"
$ corenote search "TODO" --folder Work
$ corenote search "budget" --json

Create a note

Create notes inline with a title and Markdown body, or open your $EDITOR for a full editing experience.

--title --body --folder --editor
$ corenote create --title "Shopping List"
$ corenote create --editor
$ corenote create --folder Ideas --editor

Edit a note

Open any note in your editor by fuzzy title match. Or update the title and body inline without opening an editor.

--id --title --body
$ corenote edit "shopping list"
$ corenote edit "note" --title "New Title"
$ corenote edit "note" --body "Updated."

Delete a note

Soft-delete to Recently Deleted, or permanently remove with confirmation. Use --force to skip the prompt.

--id --permanent --force
$ corenote delete "old note"
$ corenote delete "draft" --permanent
$ corenote delete "tmp" --permanent --force

Move a note

Move a note to a different folder. Find the note by fuzzy title match or internal ID.

--to --id
$ corenote move "meeting notes" --to Archive
$ corenote move 42 --id --to Work

Manage folders

List, create, rename, and delete folders. Supports nested folder hierarchies with --parent.

list create rename delete
$ corenote folder list --json
$ corenote folder create "Projects"
$ corenote folder create "Q2" --parent Projects
$ corenote folder rename "Old" --name "New"
How it works
Direct database
access. No API.
corenote reads and writes Apple Notes' SQLite database directly. Note bodies are compressed protobuf — corenote handles the decoding transparently.

Requirements
Before you start
corenote accesses the Apple Notes database directly. Two things are needed.

macOS 15+

Sequoia or later. Native Swift 6 binary, compiled for Apple silicon and Intel.

Full Disk Access

Grant your terminal Full Disk Access in System Settings > Privacy & Security.
