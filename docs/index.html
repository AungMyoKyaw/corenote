Install in seconds

Available via Homebrew. Requires macOS 15 (Sequoia) or later.

$ brew tap AungMyoKyaw/tap
$ brew install corenote

Built for the terminal

R/W

Full CRUD

List, show, create, edit, delete, and move notes. Everything you do in Notes.app, from the command line.

~/

Folder management

Create, rename, delete, and nest folders. Organize your notes the way you want.

~=

Fuzzy matching

Find notes by approximate title. Four-level matching: exact, prefix, contains, and Levenshtein distance.

md

Markdown support

Write in Markdown — corenote converts to and from Apple Notes' internal protobuf format transparently.

vi

$EDITOR integration

Create and edit notes in your preferred editor. Works with vim, nvim, nano, or any $EDITOR.

{}

JSON output

Every query command supports --json for structured output. Pipe to jq, scripts, or other tools.

All the commands you need

list
$ corenote list
$ corenote list --folder Work
$ corenote list --sort title --json
show
$ corenote show "meeting notes"
$ corenote show 42 --id
$ corenote show "my note" --raw
search
$ corenote search "deadline"
$ corenote search "TODO" --folder Work
create
$ corenote create --title "New Note"
$ corenote create --editor
$ corenote create --folder Ideas
edit
$ corenote edit "shopping list"
$ corenote edit "note" --title "New"
delete
$ corenote delete "old note"
$ corenote delete "note" --permanent
move
$ corenote move "note" --to Archive
folder
$ corenote folder list
$ corenote folder create "Projects"
$ corenote folder rename "Old" --name "New"

Requirements

corenote accesses the Apple Notes database directly — no API, no sync.

macOS 15+

Sequoia or later required. Native Swift 6 binary.

Full Disk Access

Grant your terminal Full Disk Access in System Settings > Privacy & Security.
